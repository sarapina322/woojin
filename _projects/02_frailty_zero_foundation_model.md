This project sits within a broader effort to build a **multimodal foundation model for frailty assessment** and preventive care in an aging society.

My interest here is in how diverse modalities—physiological signals, clinical information, and real-world behavioral data—can be aligned into a unified representation that supports earlier risk detection and more practical intervention design.

### My contribution focus

- multimodal representation learning for frailty-related signals
- healthcare-oriented model design for prevention rather than late-stage reaction
- translation from model outputs to actionable intervention pathways
- human-centered service thinking for real-world healthcare deployment

This page intentionally keeps the description public-facing. Internal project documents and proposal materials are **not** included in the site package.
