I am involved in the development of a **smart Foley catheter system for neurogenic bladder**, with the goal of preserving long-term bladder compliance and function through a more physiological closed-loop control strategy.

### Core components

- pressure-based signal interpretation for distinguishing meaningful bladder dynamics from artifacts
- non-invasive motor-driven pinch valve design
- ex-vivo validation workflows and preparation for clinical translation

### Related IP

- Foley Catheter Urination Control Device and Control Method Thereof  
  Korea application no. **10-2026-0006080**

What I find compelling about this project is that it demands both **engineering precision** and **clinical realism**. It is not only a device problem and not only a model problem; it is a full system design problem.
